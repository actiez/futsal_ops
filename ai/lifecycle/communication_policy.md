# Communication Policy

## Principle

All player interactions must be routed through the WhatsApp Assistant.

Admins should not handle player messages directly.

---

## Player Flow

- Player receives event notification via WhatsApp Assistant
- Player replies to WhatsApp Assistant to:
  - join
  - leave
  - check status

---

## Admin Flow

- Admin reviews events
- Admin approves event broadcast
- WhatsApp Assistant sends message to players

---

## Rule

If players message admin directly:
→ Admin should redirect them to WhatsApp Assistant

---

## Purpose

- Centralise interaction
- Ensure system consistency
- Enable AI tracking and automation
- Prevent fragmented communication
