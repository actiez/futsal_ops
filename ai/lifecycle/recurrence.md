# Recurring Event Logic

## Types

- Weekly
- Custom interval (future)

---

## Fields

- recurrence_enabled (true/false)
- recurrence_type (weekly)
- recurrence_day (Sunday)
- recurrence_time (19:00)

---

## Behavior

- When event completes → generate next event
- Link events in series (optional)

---

## Admin Controls

- Enable / disable recurrence
- Modify future events
