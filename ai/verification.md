# Verification Framework

## Step 1: Identity Match
Match WhatsApp number with registered mobile number

If no match:
→ Redirect to registration

## Step 2: Risk-Based Verification

Low Risk:
- Join / Leave
- Check status

High Risk:
- Create event
- Reports
- Custom queries

## Step 3: Verification Methods
- Username
- Event-related question
- OTP (future)

## Step 4: Trust Window
Verification valid for 15–30 minutes
