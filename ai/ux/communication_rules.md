# Communication Rules

## Core Principle

All player interactions must go through WhatsApp Assistant (PM).

---

## Group Chat

- Used ONLY for broadcast
- No actions allowed
- No replies expected

---

## Private Message (PM)

- All actions handled here:
  - join
  - leave
  - status

---

## Admin Behavior

If player messages admin directly:

Admin should reply:

"Please use the assistant to manage your game: [link]"

---

## Purpose

- Maintain system consistency
- Prevent manual bypass
- Enable full tracking and automation
