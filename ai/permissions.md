# Role & Permission Matrix

## Player
Allowed:
- Join / Leave events
- View own status and games

Restricted:
- Cannot view other users
- Cannot access reports
- Cannot create events

## Event Manager
Allowed:
- Create events
- View event-level data
- Generate reports

Restricted:
- No personal user data
- No system-wide analytics

## Admin
Allowed:
- Full event visibility
- Operational reports

## Superadmin
Allowed:
- Full system access
